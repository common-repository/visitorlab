== Description ==

VisitorLAB is an analytics solution for websites to visualize how visitor act on it.  

With a unique feature set, VisitorLAB gives you a way to understand user behavior behind the number. 

== List of features ==

* **Heatmaps** - Visitor recordings show you every action of your visitors on your website to show you how your visitors use it. So website owner can watch visitors user experience through videos.
* **Visitor Recordings** - Heatmaps about measuring visitors' click and mouse movements, which part of your web site get more attraction and wich part totally ignore by your visitors. 
* **Error Tracking** - VisitorLAB creates a detailed list of errors that occur in browsers and makes an alert for the website owners.

This plugin provides a simple installation of VisitorLAB on your WordPress site.

== Installation ==

= Simple installation =

The VisitorLAB plugin is available to install via the WordPress plugin library and you can install it directly from within your WordPress dashboard.

= Manual installation =

1. Download the VisitorLab plugin as a zip file.
2. Put the folder `visitorlab` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
